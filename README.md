# Poké Play

A fun app to show Pokémon figure using the [PokéStadium](http://www.pokestadium.com) API with beautiful background.

Live app can be access here: [pokeplay.lekosfmi.com](http://pokeplay.lekosfmi.com)

## Technology Stack
- ReactJS
- MeteorJS
- NodeJS
